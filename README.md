## HomeScreen — FamilyBusiness App

La pantalla HomeScreen es la primera vista que recibe el usuario y resume los pilares principales de la aplicación FamilyBusiness, destacando sus beneficios, testimonios de usuarios y el cta.

Está diseñada para ofrecer una experiencia visual clara, moderna y enfocada en la conversión (registro).

### Componentes incluidos
1. HeroSection
- Presentación principal de la app.
- Incluye título, subtítulo y una imagen destacada.

2. BenefitsSection
- Tarjetas con los beneficios clave de FamilyBusiness.
- Cada tarjeta muestra un ícono, título y descripción breve.

3. TestimonialsSection
- Opiniones reales de usuarios.
- Presenta nombre, comentario y a veces avatar.

4. CTASection
- Una llamada a la acción clara.
- Contiene un botón de registro para avanzar en el flujo de usuario.

### Tecnologías
- React Native
- ExpoGo
- Hooks
- StyleSheet para estilos

### Características principales
- Diseño moderno y responsivo.
- Animación simple en el botón principal.
- Tarjetas con sombras en iOS y Android.
- Scroll vertical completo.
